# Proyecto Final: Predicción de índices de calidad del aire (AQI) para distintas estaciones de monitoreo en Delhi
## Diplomatura en Ciencia de Datos y Análisis Avanzado – UTN

## Descripción
Este proyecto tiene como objetivo predecir el índice de calidad del aire (AQI) en Delhi utilizando modelos de ciencia de datos.

## Dataset
Se utilizó un dataset público extraído de Kaggle: https://www.kaggle.com/datasets/sohails07/delhi-weather-and-aqi-dataset-2025.

## Ejecución
1. Abrir el notebook en Google Colab
2. Ejecutar las celdas en orden

## Modelos
- Regresión Logística
- Random Forest
- XGBoost
- Redes Neuronales Densas (DNN)
- La evaluación se realizó utilizando métricas de accuracy, F1-score ponderado y AUC-ROC, considerando el desbalance de clases presente en los datos.
